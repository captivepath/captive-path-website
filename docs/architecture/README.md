# Architecture

## Overview
Captive Path Website is a static marketing site built with React + Vite + Tailwind CSS and deployed to Cloudflare Pages.

## Structure
- Single-page application (homepage only for now)
- Component-based architecture with section-level components
- Design tokens defined in Tailwind configuration
- CI/CD via GitHub Actions deploying to Cloudflare Pages
