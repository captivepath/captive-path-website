## Design Direction
Modern editorial systems design with restrained character.
- Adtriox clarity + Atomle structure + Captive Path brand restraint
- Display font + clean sans-serif body
- Warm mineral/stone neutrals, deep graphite text, dark teal accent
- Left-aligned hero, modular panel structure, editorial dividers
- No neon, no orbs, no centered-everything, no generic SaaS patterns

## Tech Stack
- React 19 + Vite
- Tailwind CSS 4
- Cloudflare Pages (static deployment)
- GitHub Actions for CI/CD
